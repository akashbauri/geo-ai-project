# GEO AI Project

## Overview
This project analyzes a webpage and generates JSON-LD structured data to improve AI search visibility (GEO).

## Tech Stack
- FastAPI (Backend)
- Groq API (LLM)
- HTML/CSS/JS (Frontend)

## Features
- Web scraping
- AI-based schema generation
- Simple UI for demo

## Deployment
- Frontend: Vercel
- Backend: Render

## Key Decision
Used hybrid approach:
- Rule-based scraping for reliability
- Groq LLM for intelligent schema generation
